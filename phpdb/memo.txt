

# 1. CTFdのソースコードをダウンロード
git clone https://github.com/CTFd/CTFd.git
cd CTFd
# 2. 必要なデータベース等のイメージをダウンロード（Pull）
docker compose pull
# 3. 【超重要】CTFd本体のイメージを「ビルド」して作成する
# （これをやらないとオフラインでPythonのパッケージがDLできずエラーになります）
docker compose build
# 4. 準備できたDockerイメージの一覧を確認する
docker images
