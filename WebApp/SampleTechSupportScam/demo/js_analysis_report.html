Tech Support Scam JS 解析レポート

assets/js/ ディレクトリ内に含まれる各JavaScriptファイルの挙動および関数の詳細な解析結果です。これらのファイルはテクニカルサポート詐欺において、ユーザーの不安を煽り、ブラウザの操作を妨害するために使用されています。

11gfdjuef.js
ブラウザクラッシュ・フリーズ攻撃

全体挙動: ユーザーがページを閉じようとしたり、別のページへ移動しようとした瞬間に、ブラウザのCPUリソースを強制的に枯渇させてフリーズさせる極めて悪質なスクリプトです。

  • window.addEventListener('beforeunload' / 'unload'): ページから離れる直前・瞬間のイベントを検知します。
  • workerBomb(): Web Worker（バックグラウンド処理）を使用して無限ループ(while(true))による乱数計算を実行するスクリプトを生成し、それを1兆個(1000000000000)生成しようとします。この関数自体が20回連続で呼び出されるため、瞬時にPCをフリーズさせます。
  • startWatchdog(): 本来はブラウザがフリーズした際にWorkerを強制終了させる安全装置ですが、コメントアウトされて無効化されています。
  • terminateWorkers(): 生成したWorkerを強制終了する関数ですが、現在は呼び出されません。
04shesc1.js
キーボード操作の無効化

全体挙動: システムレベルでのキーボード操作をブロックし、ユーザーがショートカットキーなどでブラウザを操作・終了できないようにします。

  • navigator.keyboard.lock(): ブラウザのFullscreen APIなどと連携して、OSレベルのキーボードショートカット（Escキーなど）をロックします。
  • document.onkeydown = function (e) { return false; }: ページ上でのすべてのキー入力を無効化します。
05sdghdf.js
右クリック禁止・特定キーブロック・UI制御

全体挙動: コンテキストメニュー（右クリック）を禁止し、リロード（F5）やキャンセル（ESC）をピンポイントで無効化します。また、クリック時に各種警告UIを表示します。

  • mousehandler() / mischandler(): 右クリックを検知し、return false で無効化します。
  • win_onkeydown_handler(): 押されたキーのキーコードを判定し、116 (F5) や 27 (ESC) の場合にイベントをキャンセルします。
  • UI制御: #mycanvas などをクリックした際に、隠されている警告ポップアップ (#welcomeDiv, #poptxt, #footer) を表示します。
08dgsg3d.js / 02dgdsg3d.js
強制フルスクリーン・音声再生・離脱警告

全体挙動: 画面の任意の場所をクリックした際に強制的にフルスクリーンモードにし、偽の警告音をループ再生します。2つのファイルは全く同じ内容です。

  • toggleFullScreen(e): ブラウザのFullscreen API (requestFullScreen等) を呼び出し、ページを全画面表示にします。
  • 音声再生: <audio> 要素を動的に生成し、外部のmp4/wavファイルを読み込みます。画面クリック時に e.play() で再生を開始し、終了時にループ再生させます。
  • 離脱警告: beforeunload イベントで「保存していない変更が失われます」という標準の警告ダイアログを表示させ、ページを閉じるのを躊躇させます。
13dugfjdf.js / 07sdgsg4.js
Exit Intent (離脱検知) ポップアップ

全体挙動: マウスカーソルがブラウザの画面外に出たことを検知し、ユーザーがタブを閉じようとしていると判断して警告ポップアップを表示します。2つのファイルは同じ内容です。

  • addEvent(document, 'mouseout', ...): マウスがドキュメント外に出た際（evt.toElement == null）に、警告UIである .lightbox をスライドダウンで表示します。
  • クリック制御: ポップアップの閉じるボタンや背景をクリックした際にポップアップを隠します。
09sgsgsfr.js
IPアドレス・位置情報取得

全体挙動: 外部のIP情報取得APIを叩き、アクセス元のIPアドレスや地域情報を画面に表示して「個人情報が特定されている」と脅します。

  • XMLHttpRequest: https://ipwho.is/?lang=en に対して非同期リクエストを送信します。
  • DOM操作: 取得したJSONデータからIPアドレス、都市、国、プロバイダ(ISP)を抽出し、document.getElementById() を用いて画面上の特定の要素にテキストとして埋め込みます。
03fgsskryeivh.js
警告音の追加再生

全体挙動: 画面の特定の要素をクリックした際に、さらに複数の警告音を鳴らします。

  • audioElement.play(): ローカルの dgdh1.wav と、外部URLの音声ファイルの2種類のAudio要素を作成し、クリックイベントに紐付けて再生・ループさせます。
01d1fgshfddfg.js
プログレスバー・Favicon点滅アニメーション

全体挙動: スキャンが進行中であるように見せかけるアニメーション処理と、タブのアイコンを点滅させる処理を行います。

  • setInterval(): 100ミリ秒ごとに変数を加算し、プログレスバー(#dynamic)の幅(width)とテキストを0%から100%まで変化させます。
  • a.fn.countTo(): 数値がカウントアップしていくようなアニメーションを実装するjQueryプラグインです。
  • Favicon点滅: 1秒ごとに #favicon の href 属性を書き換え、アイコンが点滅しているように見せかけます。
  • フルスクリーン化: ここでも画面クリック時に requestFullScreen を呼び出す処理が重複して記述されています。
jquery-1.4.4.min.js / 12dgdur.js (jQuery 2.1.3)
ライブラリ (jQuery)

全体挙動: 他のスクリプトでDOM操作やイベント処理を簡単に行うために読み込まれている標準的なjQueryライブラリです。複数のスクリプトのツギハギで作られているため、バージョン違いが2つ読み込まれています。

06hshs.js
不要ファイル (ゴミファイル)

全体挙動: JavaScriptとしての機能はありません。中身はHTMLのコードがコメントアウトされたものと、空のHTMLテンプレートです。詐欺サイトを自動ツール等でコピーした際の誤配置と考えられます。
